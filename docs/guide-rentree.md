# Guide de rentrée — Carnet EPS

> À dérouler **fin août / début septembre**, en ~30 minutes. Objectif : arriver au premier cours avec l'appel prêt en deux taps.

## 1. Archiver l'année passée (5 min) — si l'app a déjà servi

1. **Plus → Sauvegarde → Télécharger la sauvegarde** (avec pièces jointes).
2. Renommer le fichier `carnet-eps_archive_2025-2026.json` et le ranger en lieu sûr (PC + une copie ailleurs). Conservation conseillée : 1 an scolaire.
3. **Plus → Sauvegarde → Effacer toutes les données** (une sauvegarde de sécurité est re-téléchargée automatiquement avant, double confirmation).

## 2. Régler l'année (2 min)

**Plus → Réglages** :
- Année scolaire : `2026-2027`
- Établissement : vérifier
- Thème : au goût du jour 🌞/🌙

## 3. Importer les élèves depuis Pronote (10 min)

Pour **chaque classe** :
1. Dans le client Pronote : Ressources ▸ Élèves ▸ liste de la classe → copier le tableau (ou exporter en CSV).
2. Dans l'app : **Élèves → Importer depuis Pronote (CSV)** → coller (ou choisir le fichier) → **Analyser**.
3. Vérifier le mapping des colonnes (Nom, Prénom, Né(e) le, Sexe, Classe) → **Importer**.
4. Les classes se créent toutes seules depuis la colonne Classe ; les doublons sont ignorés (on peut relancer sans risque).

💡 Astuce : si l'export contient toutes vos classes d'un coup, un seul import suffit.

## 4. Saisir l'EDT (5 min)

**EDT** :
1. Si votre emploi du temps alterne sur deux semaines : renseigner « **Un lundi de semaine A** » (n'importe quel lundi d'une semaine A, par ex. le lundi de la rentrée si c'est une semaine A).
2. **+ Ajouter un créneau** pour chaque cours : jour, heures, classe, semaine (Toutes/A/B), installation.

⚠ Rappel : l'alternance suit la parité des semaines calendaires — vérifier après les premières vacances que A/B ne s'est pas inversé (sinon, décaler le lundi de référence d'une semaine).

## 5. Créer les premières séquences (5 min)

**Plus → Séquences → + Nouvelle séquence** pour chaque classe : APSA du premier cycle, champ d'apprentissage, dates (début de cycle → vacances), nombre de séances prévues.

Pas besoin de créer les séances : **l'accueil propose la séance du jour automatiquement** à l'heure du cours.

## 6. Vérifier (3 min)

- [ ] L'accueil affiche le bon cours au bon créneau (ou « Pas de cours aujourd'hui » un jour off)
- [ ] Ouvrir une classe : effectif complet, accents corrects
- [ ] Faire un appel de test sur une séance fictive… puis supprimer la séance (Plus → Séquences → la séquence → ✕)
- [ ] **Exporter une première sauvegarde** de l'année neuve

## Le jour J (premier cours)

1. Ouvrir l'app → la carte « En ce moment » affiche la classe.
2. Toucher **« Créer la séance 1/X et faire l'appel »**.
3. Faire l'appel au pouce. C'est tout.

## Au fil de l'année — les bons réflexes

- **Exporter une sauvegarde chaque semaine** (vendredi soir, 10 secondes) — c'est la seule protection contre la perte de l'appareil.
- Certificat reçu → **Plus → Inaptitudes → + Nouvelle** avec photo : l'élève sera signalé à l'appel tout seul.
- Fin de cycle → saisir les notes dans l'app → **« Copier pour Pronote »** au bureau.
- L'accueil vous rappelle le reste (inaptitudes qui expirent, seuils de tenue, notes non remontées).
